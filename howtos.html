Frequently Asked Questions

How to add link to agenda in deck?

Use <xp-deck back="../"> to add a back link before first slide.

How to alter git history?

In short:
$ git rebase -i --root
$ git commit -a && git rebase --continue
More details in Rewriting history section of xpla.org/git/help.

How to change the page that is displayed in preview?

Create & commit /_xp-preview-file with path to index.html.
Example: $ echo "dist/index.html" > _xp-preview-file

How to modify runner when generating slides?

Create & commit /_xp-runner file with runner name inside it.
Example: $ echo "webpack" > _xp_runner

How to display files in a tree?

Create & commit empty /_xp-tree file.

My syntax is not highlighted / comments are not working

Let us know! We support limited number of highlighted files, but it's not an issue to add some more.
