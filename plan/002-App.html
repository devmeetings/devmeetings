Let's write an app!

A core part of Devmeeting is writing an app

On every Devmeeting participants should write full-featured (yet limited) app from scratch, completely on their own.

After choosing a topic you should start thinking on what app participants will write.

Why app? It should be something tangible, something to show to their friends (or best: gradma) after the workshop. With many "visual assertions" indicating that the task is actually done - it should be very easy to test if task has been done correctly.
